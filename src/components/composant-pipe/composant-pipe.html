Les pipes sur angular:

Les pipes sur angular s'utilise avec le ' | ' Les pipes peuvent prendre des paramètres (au final les pipes sont des fonctions). Chacun des paramètres utilisés par une pipe est separé par un ':'

Les pipes angular :

  • Le pipe date
  • Le pipe currency
  • Le pipe json
  • Les pipes lowercase et uppercase
  • Le pipe number
  • Le pipe percent
  • Le pipe async

Exemple avec pipe Date:

Sans le pipe : {{ myDate }} / {{myDate2}}

avec le pipe, sans paramètre :{{ myDate | date}}

avec param 'shortDate': {{ myDate2 | date: 'shortDate'}}

avec param 'fullDate': {{ myDate2 | date: 'fullDate'}}

avec param 'short': {{ myDate2 | date: 'short'}}

avec param 'full': {{ myDate2 | date: 'full'}}

{{ myDate2 | date:'dd/MM/yyyy' }}

{{ myDate2 | date:'EEEE, d MMMM y' }}

Exemple avec pipe Currency:

{{ money | currency:'EUR':'symbol':'1.0-0' }}

{{ money | currency:'EUR':'symbol':'1.0-2' }}

{{ money | currency:'EUR':'symbol':'1.2-2' }}

{{ money | currency:'USD':'symbol':'1.0-0' }}

{{ money | currency:'EUR':'symbol-narrow':'1.0-0' }}

{{ money | currency:'EUR':'nom personnalisé':'1.2-2' }}
